RoomAmp2 v0.5

RoomAmp2 control is a software that enables you to take over control of a Linn RoomAmp 2.
It is capable of reading and displaying status values from the Roomamp as well as

  • set the input channel
  • adjust the volume in the main room
  • adjust the balance
  • mute/unmute the device
Pls visit our website for installation instructions and new releases.
